# Spring REST using Spring Boot - Hands-on 1

## Objective

Create a Spring Boot Web project using Maven and understand the generated project structure.

## Technologies Used

- Java 21
- Spring Boot 3
- Maven
- Spring Web
- Spring Boot DevTools

## Project Structure

```
src/
 ├── main/
 │    ├── java/
 │    └── resources/
 │
 └── test/
      └── java/
```

## Key Concepts Learned

- Spring Initializr
- Maven Project Structure
- Embedded Tomcat
- SpringApplication.run()
- @SpringBootApplication
- Auto Configuration
- Component Scanning
- Dependency Management
