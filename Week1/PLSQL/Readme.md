# PL/SQL Exercises

## Exercise 1 - Control Structures

### Scenarios Implemented

1. Apply a 1% loan interest discount for customers above 60 years.
2. Promote customers with balance greater than 10000 to VIP.
3. Print reminder messages for loans due within the next 30 days.

### Concepts Used

- FOR LOOP
- IF-THEN
- UPDATE
- SELECT
- COMMIT
- DBMS_OUTPUT.PUT_LINE

# Exercise 3 - Stored Procedures

### Procedures Implemented

1. ProcessMonthlyInterest
   - Adds 1% monthly interest to all savings accounts.

2. UpdateEmployeeBonus
   - Increases employee salary based on department and bonus percentage.

3. TransferFunds
   - Transfers money between two accounts after checking sufficient balance.

### Concepts Used

- CREATE PROCEDURE
- Parameters (IN)
- IF-ELSE
- UPDATE
- SELECT INTO
- COMMIT
- DBMS_OUTPUT.PUT_LINE
